# Documentation

Guides for deploying and operating Paperclip on Google Cloud Run.

## Table of Contents

### For board members and operators

- **[Board Operations Guide](board-operations-guide.md)** — non-technical guide for managing agents, goals, issues, hiring, 3rd party integrations, routines, and cost optimization. No coding required.

### For platform engineers

- **[Cloud Deployment Guide](cloud-deployment-guide.md)** — technical field guide for deploying Paperclip to GCP Cloud Run. Covers the 7 workarounds needed to bridge Paperclip's local-first architecture to stateless compute, with Terraform code and architecture diagrams.

### Combined (publishable)

- **[Paperclip Cloud Operations Guide](paperclip-cloud-operations-guide.md)** — combined guide covering board operations, 3rd party integrations, example workflows (social media campaigns, competitive research, Remotion video), and skill installation. Also available as [.docx](paperclip-cloud-operations-guide.docx) for sharing outside GitHub.
